# Day 07 — Strings

## Objectives
- Master the most-used `std::string` methods
- Convert between strings and numbers
- Split and parse strings with `std::istringstream`

## Concepts Covered
- Construction, `.length()`, `.empty()`
- `[]` vs `.at()` (bounds-checked access)
- Concatenation with `+` and `+=`
- `.substr()`, `.find()`, `.rfind()`, `std::string::npos`
- `.replace()`, `.erase()`, `.insert()`
- `<` comparison (lexicographic ordering)
- `std::transform` + `::tolower/toupper` for case conversion
- `std::stoi`, `std::stod`, `std::to_string`
- Splitting with `std::istringstream` and `getline(..., delim)`
- `.c_str()` for C library interop
