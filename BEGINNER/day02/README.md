# Day 02 — Operators, Expressions, Type Casting, const

## Objectives
- Master all arithmetic, relational, and logical operators
- Understand integer vs floating-point division
- Apply `static_cast<>` correctly (C++ preferred over C-style cast)
- Use `const` to protect values from accidental change
- Use `<cmath>` for sqrt, pow, abs

## Concepts Covered
- `+  -  *  /  %` arithmetic, `+=  -=  *=  /=  %=` compound assignment
- `++  --` prefix vs postfix
- `<  >  <=  >=  ==  !=` relational, `&&  ||  !` logical
- `&  |  ^  <<  >>` bitwise operators
- `(int)x` vs `static_cast<int>(x)` — always prefer the latter in C++
- `const` variables vs `#define` macros (prefer `const`)
- `<cmath>`: `std::sqrt`, `std::pow`, `std::abs`
