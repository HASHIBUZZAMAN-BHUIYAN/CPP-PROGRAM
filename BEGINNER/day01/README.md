# Day 01 — C++ Basics

## Objectives
- Understand what C++ is and how it differs from C
- Write, compile, and run your first C++ program
- Use `std::cout` and `std::cin` for output/input
- Declare variables with correct types
- Understand the role of `#include` and namespaces

## Concepts Covered
- `#include <iostream>`, `#include <string>`
- `std::cout`, `std::cin`, `std::endl`
- Primitive types: `int`, `double`, `float`, `char`, `bool`
- Variable declaration and initialization
- `using namespace std;` — what it does and when to avoid it
- The `main()` function and return codes
