# Day 09 — OOP I: Classes and Objects

## Objectives
- Design classes with proper encapsulation (private data, public interface)
- Write constructors, destructors, and member functions
- Understand `this` pointer and static members
- Build reusable classes that model real-world objects

## Concepts Covered
- `class` vs `struct` (same except default access: struct=public, class=private)
- `private`, `public`, `protected` access specifiers
- Constructor initialization list (`: member(value)`) — prefer over body assignment
- Default constructor, parameterized constructor
- Destructor `~ClassName()` — cleanup
- `const` member functions — don't modify the object
- `static` data members and functions — shared across all instances
- `this` pointer — pointer to the current object
