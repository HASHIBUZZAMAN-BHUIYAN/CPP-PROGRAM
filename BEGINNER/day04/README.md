# Day 04 — Loops

## Objectives
- Control repetition with for, while, and do-while
- Use break/continue to alter loop flow
- Iterate collections with range-based for (C++11)
- Build nested loops for 2D output

## Concepts Covered
- `for (init; condition; update)` loop
- `while (condition)` loop
- `do { } while (condition)` — runs at least once
- `break` — exits the loop immediately
- `continue` — skips to the next iteration
- Range-based `for (auto& x : container)`
- `auto` type deduction
