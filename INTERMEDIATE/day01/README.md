# INTERMEDIATE Day 01 — Templates

## Objectives
- Write function and class templates that work with any type
- Understand template type deduction
- Write explicit template specializations for specific types
- Use non-type template parameters

## Concepts Covered
- `template<typename T>` function template
- Template type deduction (compiler figures out T from the argument)
- `template<>` explicit specialization
- `template<typename T, int N>` non-type template parameter
- Class templates: `Stack<T>`, `Pair<T,U>`
- Template with iterator parameters
- C++17 structured bindings for pair/tuple return
